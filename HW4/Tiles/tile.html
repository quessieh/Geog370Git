Georeferencing Images

This project displays a Georeferenced map with point layers.

I was able to georeference this image in a multitude of steps. To begin, I downloaded data to my computer that consisted of a High Resolution Orthoimagery and a topographical map that had not been georeferenced since 1993. I uploaded these images to QGIS where I was able to create points within a Georeference system, which allowed me to sync the two maps up perfectly. I then created an html link for the data and created an hierarchy of files to upload to Visual Studio where I was able to display the interactive map below.
